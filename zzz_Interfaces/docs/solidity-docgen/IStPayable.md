## <span id="IStPayable"></span> `IStPayable`

Controls payment flows through the contract's default payable method


Only applicable for CASFHLOW-type contract instances

- [`onlyOwner()`][IOwned-onlyOwner--]
- [`onlyWhenReadWrite()`][IOwned-onlyWhenReadWrite--]
- [`fallback()`][IStPayable-fallback--]
- [`setIssuerValues(uint256 wei_currentPrice, uint256 cents_currentPrice, uint256 qty_saleAllocation)`][IStPayable-setIssuerValues-uint256-uint256-uint256-]
- [`setReadOnly(bool _readOnly)`][IOwned-setReadOnly-bool-]

### <span id="IStPayable-fallback--"></span> `fallback()` (external)

Handles (1) subscriber payments and (2) issuer payments (bond interest payments & principal repayments, and equity dividend payments)



### <span id="IStPayable-setIssuerValues-uint256-uint256-uint256-"></span> `setIssuerValues(uint256 wei_currentPrice, uint256 cents_currentPrice, uint256 qty_saleAllocation)` (external)

Sets current values for the issuance


(wei_currentPrice XOR cents_currentPrice) -- price per token: write-once for BOND, write-many for EQUITY
qty_saleAllocation                        -- sale allocation: must be >= current qty subscribed, and <= total monobatch size
